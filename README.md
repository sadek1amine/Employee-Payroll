# 💼 Employee Payroll System

A modern **Employee Payroll Management System** built with Next.js, Prisma, and Tailwind CSS.  
This system helps manage employees, calculate salaries, track attendance, and generate reports.

---

## 🚀 Features

### 👨‍💼 Employee Management
- Add / Edit / Delete employees
- Store employee details (name, email, position, salary)
- Assign employees to branches

### 💰 Payroll System
- Automatic salary calculation
- Base salary + bonus + deductions
- Overtime support
- Tax calculation
- Net salary generation

### ⏱️ Attendance Tracking
- Track employee attendance
- Mark Present / Absent / Late
- Link attendance with payroll

### 📊 Reports
- Payroll reports
- Attendance reports
- Export-ready structure (PDF/Excel ready)

### 🔐 Authentication (Future-ready)
- Login system structure ready
- Role-based access (Admin / HR / Manager)

---

## 🧠 Tech Stack

- ⚛️ Next.js (App Router)
- 🟦 TypeScript
- 🎨 Tailwind CSS
- 🗄️ Prisma ORM
- 🐘 PostgreSQL (or MySQL)
- ⚡ React Hooks
